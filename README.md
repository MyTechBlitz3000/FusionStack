# NovaFusionDB

A realistic hybrid full-stack database platform built using:

- 🐍 Python
- 🗄️ SQLite3
- 🍎 SwiftUI
- 🌐 HTML/CSS/JavaScript
- ⚡ FastAPI
- 🔄 GitHub Actions

NovaFusionDB combines native frontend development, backend APIs, local databases, and web dashboards into a single modular repository.

---

# Features

- SQLite3 database system
- FastAPI backend API
- Native SwiftUI frontend
- Web dashboard
- REST API support
- Realistic project architecture
- GitHub CI/CD workflows
- Modular codebase
- Local-first development
- Cross-platform support
